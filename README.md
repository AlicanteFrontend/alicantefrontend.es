# Alicante Frontend

Sitio web oficial de [Alicante Frontend](https://alicantefrontend.es), la comunidad de desarrollo web frontend de Alicante.

> El ecosistema Frontend es un caos y ¡nos encanta! Si quieres aprender y compartir sobre desarrollo web orientado al Frontend has llegado al sitio correcto.

## Cómo contribuir

Este sitio está construido con [Astro](https://astro.build) y desplegado en GitHub Pages. Las contribuciones son bienvenidas: corregir erratas, mejorar contenido, arreglar bugs o proponer nuevas secciones.

### Requisitos

- [Node.js](https://nodejs.org) >= 22.12.0
- [pnpm](https://pnpm.io) (gestor de paquetes utilizado en el proyecto)

### Puesta en marcha

```bash
# 1. Haz un fork del repositorio y clónalo
git clone https://github.com/<tu-usuario>/alicantefrontend.es.git
cd alicantefrontend.es

# 2. Instala las dependencias
pnpm install

# 3. Arranca el servidor de desarrollo
pnpm dev
```

### Flujo de trabajo

1. Crea una rama desde `main` con un nombre descriptivo (`fix/typo-home`, `feat/eventos`).
2. Realiza tus cambios y verifica localmente con `pnpm dev` que todo funciona.
3. Asegúrate de que `pnpm build` termina sin errores.
4. Abre un Pull Request hacia `main` describiendo el cambio y, si procede, adjunta capturas.

Para cambios grandes, se recomienda abrir un issue para comentar la propuesta antes de invertir tiempo en la implementación.
